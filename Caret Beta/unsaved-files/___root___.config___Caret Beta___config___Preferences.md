# Preferences

Edit and save to update preferences:

```yaml
font: "A"
fontSize: 18
codeFont: "Roboto Mono"
codeFontSize: 17
lineHeight: 1.6
lineWidth: 96
tabSize: 2
useLargeFontInHeadings: true
```

Want to see another option? Let us know over [Twitter](https://twitter.com/careteditor) or [email](mailto:contact@caret.io).
